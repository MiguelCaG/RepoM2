# Flappy Bird

## Proyecto desarrollado por Miguel Carmona Garrido

### Versión de Unity:

Unity 2022.3.62f

### Controles:

Utiliza el **click izquierdo** o la tecla **espacio** para hacer al pájaro saltar.

Utiliza la tecla **escape** para pausar el juego y abrir el menú de opciones de audio.

### Crear build jugable:

Ir a **File>Build Settings...** o utilizar el comando **Ctrl+Shift+B**. Pulsar el botón **Player Settings...** e ir a **Player>Resolution and Presentation** y establecer las siguientes opciones:
* Fullscreen Mode -> Windowed
* Default Screen Width -> 1280
* Default Screen Height -> 720
